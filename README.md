# Product Roadmap Dashboard — 2025

An interactive roadmap dashboard built to demonstrate cross-functional 
PM planning across Design, Engineering, Data, and Growth teams.

## Live demo
👉 https://varikutiteja.github.io/product-roadmap-2025/  

## What this shows
- Quarterly initiative planning across 4 workstreams
- Risk tracking and status visibility (on track / at risk / completed)
- Stakeholder-ready format for QBRs and leadership reviews
- Cross-functional ownership with clear due dates and priority levels

## PM decisions behind this roadmap
- Q1 focuses on retention fundamentals before scaling growth spend
- At-risk items are surfaced explicitly — not hidden in status fields
- Each initiative is tied to a named owner to enforce accountability
- Q3/Q4 are intentionally lighter to allow for mid-year reprioritization
